# 电路综合设计实验
---
## 项目介绍
基于Visual C++进行STM32开发，包含四个实验：放大器非线性失真研究装置、
简易信号分析仪、程控直流稳压源、火焰检测

## 实现原理

- 先将常用的模块进行分装，如 LED.h , PWM.h 等
- 然后在一个文件 experiment 中定义四个实验需要的类
- 最后在主程序 play.cpp 的 main 函数中实例化四个类并调用方法实现
